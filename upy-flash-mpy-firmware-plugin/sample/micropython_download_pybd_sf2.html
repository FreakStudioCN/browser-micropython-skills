Installation instructions

Pyboard-D via mboot. You can enter the mboot DFU bootloader by executing machine.bootloader() at the MicroPython REPL.

Alternatively, hold down the USR button, press and release the RST button, and continue holding USR until the LED indicates mboot mode.

Use dfu-util to write the DFU firmware file after the board enters DFU mode.

dfu-util -a 0 -D PYBD_SF2-DATE-VERSION.dfu
